A Review of HTML

HTML is really a very simple scripting language. It consists of ordinary text, with elements that are enclosed by "<" and ">" characters.
The best part about HTML is that once we know how to efficiently use basic elements, we can look up how to use all of the others!

Links

Links are very useful, and help users navigate to new pages, both within and outside of the current website.
For instance, they can be used to direct users to Google's Homepage. Emails can also be easily sent through the use of links.
They can even bring the user to a certain place within a webpage.

Images

Sometimes we want to go ahead and align pictures to the left of a line of text. Using the "align" attribute, with a value of "left" is a perfect way to do this. You could also align the picture to the right of text by replacing the value in the "align" attribute to "right".
